# MAZOU — 60 Second Video Script (v2)

**Format:** Founder direct-to-camera + screen captures of dashboard
**Tone:** Calm conviction. No hype. Every claim is something you can defend.

---

## THE SCRIPT

**[0:00–0:08] HOOK — Founder to camera, tight frame**

"There's a cost line growing inside every software company that most founders haven't noticed yet. It's not headcount. It's not cloud. It's AI. And right now, if you're building in Africa, you have zero infrastructure to manage it."

**[0:08–0:18] THE PROBLEM — Quick cuts: multiple API dashboards, declined card, billing pages**

"You're juggling three, four, five different AI providers. Each one has its own API, its own pricing, its own billing in dollars. You're paying through cards that get declined, converting Naira at terrible rates, routing every request to the same expensive model because you don't have time to do anything smarter."

**[0:18–0:28] THE PATTERN — Text on screen: AI SPEND IS THE NEW CLOUD SPEND**

"We've seen this exact pattern before. Remember when the Naira collapsed and every AWS bill tripled overnight? AI spend is following the same trajectory, on a faster timeline. It starts small. Then usage grows. Then one day the CTO looks at the bill and realises most of it was waste."

**[0:28–0:42] THE SOLUTION — Mazou dashboard on screen, smooth scroll through features**

"Mazou. One API to every model — OpenAI, Anthropic, Google, African language models — single endpoint. Fund your account in Naira. Our smart routing engine analyses every request and sends it to the cheapest model that meets your quality bar. Customers save 15 to 40 percent. And the dashboard shows you everything — spend by feature, by model, by agent — in real time."

**[0:42–0:52] THE BIGGER PICTURE — Dashboard showing agent services alongside LLM providers**

"But we're not just an API gateway. As your agents need wallets, web search, commerce — those services plug into the same Mazou interface. Same billing, same visibility. We become the layer you can't rip out because we have your complete cost picture."

**[0:52–0:58] THE CLOSE — Founder back to camera**

"Every technology wave creates its infrastructure companies. Mobile money created M-Pesa. Online payments created Paystack. AI is creating Mazou."

**[0:58–1:00] END CARD**

*MAZOU*
*One API. Every model. Pay in Naira. Spend less.*
*mazou.dev*

## PRODUCTION NOTES

- **Total runtime:** 60 seconds sharp
- **Word count:** ~230 words (natural pace, no rushing)
- **B-roll needed:** Mazou dashboard screencast, terminal API call, Naira wallet funding, smart routing animation
- **Music:** Minimal electronic, subtle build, drops out at 0:52 for the close
- **Captions:** Mandatory — large, bold, 2-3 words per frame for social
- **Aspect ratios:** Film 16:9, crop to 9:16 for Reels/Shorts with text overlays centred
- **Key visual moments:**
  - 0:18 — "AI SPEND IS THE NEW CLOUD SPEND" full-screen text hit
  - 0:28 — Dashboard reveal (hero shot)
  - 0:52 — "AI is creating Mazou" (founder conviction shot, closer frame)
- **What changed from v1:** Removed "tokens are the new currency" framing. Every claim is now factual and defensible. No metaphors that can be challenged. The cloud parallel does the heavy lifting because it actually happened.
