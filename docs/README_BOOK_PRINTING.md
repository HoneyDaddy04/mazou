# The Busy Little Bug - Book Printing Guide

## Step 1: Save the Image
1. Right-click on the book image you showed me
2. Select "Save Image As..."
3. Save it in this folder: `c:\Users\awulu\Documents\Main Development Projects\mazou\`
4. Name it: `busy_little_bug.png` (or .jpg)

## Step 2: Run the Script
Open a terminal in this folder and run:
```bash
python create_book_pdf.py
```

## Step 3: Print Your Book
1. Open the generated PDF: `The_Busy_Little_Bug_Book.pdf`
2. Print settings:
   - **Pages per sheet**: 1 (each page prints separately)
   - **Paper size**: Letter (8.5" x 11")
   - **Quality**: Best/High quality
   - **Color**: Color (not black & white)

## Book Assembly Options

### Option A: Simple Stapled Book
1. Print all 6 pages
2. Stack them in order
3. Fold in half vertically
4. Staple along the fold

### Option B: Individual Pages
1. Print all 6 pages
2. Punch holes on the left side
3. Use a binder or rings

### Option C: Professional Binding
1. Print all pages
2. Take to a print shop for spiral binding or proper book binding
