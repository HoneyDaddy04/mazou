Quick Start

Get up and running with Mazou in under 5 minutes. Our API is OpenAI-compatible, so you can use existing SDKs.

Python

from openai import OpenAI

client = OpenAI(
    base_url="https://api.mazou.ai/v1",
    api_key="mz_live_your_key_here"
)

response = client.chat.completions.create(
    model="gpt-5",
    messages=[{"role": "user", "content": "Hello!"}]
)

JavaScript

import OpenAI from "openai";

const client = new OpenAI({
    baseURL: "https://api.mazou.ai/v1",
    apiKey: "mz_live_your_key_here"
});

const response = await client.chat.completions.create({
    model: "claude-sonnet-4.6",
    messages: [{ role: "user", content: "Hello!" }]
});

cURL

curl https://api.mazou.ai/v1/chat/completions \
  -H "Authorization: Bearer mz_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-5","messages":[{"role":"user","content":"Hello!"}]}'

Authentication

All API requests require a valid API key passed via the Authorization header:

Authorization: Bearer mz_live_your_key_here

Keys prefixed with mz_live_ are production keys that debit your wallet. Keys prefixed with mz_test_ are test keys that log usage but skip billing.

Chat Completions

Create a chat completion. This endpoint is fully compatible with the OpenAI Chat Completions API.

POST /v1/chat/completions

{
  "model": "gpt-5",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "What is Lagos?"}
  ],
  "temperature": 0.7,
  "max_tokens": 1000,
  "stream": false
}

Additional Parameters

ParameterTypeDescription
tagstringLabel for cost tracking (e.g., "support-bot")
budgetstring"low", "medium", "high" - influences routing
routestring"auto" enables smart routing

Models

List all available models:

GET /v1/models

Response:
{
  "data": [
    {"id": "gpt-5", "provider": "openai", "pricing": {...}},
    {"id": "claude-sonnet-4.6", "provider": "anthropic", ...},
    {"id": "yarngpt", "provider": "african", "is_african": true, ...}
  ]
}

Streaming

Set "stream": true to receive Server-Sent Events:

const stream = await client.chat.completions.create({
    model: "gpt-5",
    messages: [{ role: "user", content: "Hello!" }],
    stream: true
});

for await (const chunk of stream) {
    process.stdout.write(chunk.choices[0]?.delta?.content || "");
}

Smart Routing

Set model: "auto" to let Mazou pick the optimal model based on your query complexity, language, and budget preference.

response = client.chat.completions.create(
    model="auto",
    messages=[{"role": "user", "content": "Bawo ni?"}],
    extra_body={"budget": "low", "tag": "yoruba-chat"}
)
# Routes to YarnGPT (Yoruba detected, lower cost)

Routing rules can be configured in the dashboard under Smart Routing → Rules.
